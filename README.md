## Available Scripts Clonning Project

In the project directory, you can run:

***step 1***
```
$ git clone `url_this_project`
```

***step 2***
```
$ nano .env
DB_USERNAME=
DB_PASSWORD=
DB_HOST=
DB_PORT=
DB_DATABASE=
```

***step 3***
```
$ sudo docker-compose up -d --build --no-deps
```
